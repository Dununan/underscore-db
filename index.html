Try it!

Open a console and copy/paste one of these statements.

console.log(db); // to view database initial state

_.get(db.posts, 1);

_.insert(db.posts, {body: 'new post', published: false});

_.update(db.posts, 1, {published: true});

_.updateWhere(db.posts, {published: false}, {published: true});

_.remove(db.posts, 1);

_.removeWhere(db.comments, {postId: 1});

Query

Using Underscore or Lo-Dash functions, you can create complex queries.
For example, this one will return the titles of the five most viewed post. 

_(db.posts)
  .where({published: true})
  .sortBy('views')
  .first(5)
  .pluck('title')
  .value();

Persist

Persistance is done using LocalStorage. 

_.save(db);

db = _.load();
